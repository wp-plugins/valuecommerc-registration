== Description ==

Valuecommerce Site Registration can be displayed only by inputting necessary attestation tag to the management screen when cooperating on the blog screen. 

The attestation tag cannot be input from a usual blog screen. Moreover, the edit of the theme etc. are troublesome. It is a plug-in to solve it. 
Valuecommerce in Japan only.

== Requirements ==

* WordPress 2.9 or later
* PHP 5.0 or later


== Installation ==

1. Unzip the plugin archive and put `valuecommerce-registration` folder into your plugins directory (`wp-content/plugins/`) of the server. 
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin at "valuecommerce-registration" menu at the WordPress admin panel. Please input attestation tag.
